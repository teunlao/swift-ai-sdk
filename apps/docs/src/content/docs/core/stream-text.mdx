# Stream Text

Produce live token streams and events. Swift mirrors Vercel AI SDK `streamText` semantics.

## Basic streaming

```swift
import SwiftAISDK
import AISDKProvider

let stream = try await streamText(
  model: model,
  input: "Name 3 release checklist items"
)

for try await event in stream {
  switch event {
  case let .textDelta(delta):
    print(delta, terminator: "")      // token piece
  case let .toolCall(call):
    print("\n[tool call]", call)
  case let .toolResult(result):
    print("\n[tool result]", result)
  case let .reasoningDelta(delta):
    print("\n[reasoning]", delta)
  case let .error(err):
    print("\n[error]", err.localizedDescription)
  case .finish(let summary):
    print("\n\nDone. Tokens:", summary.usage.totalTokens)
  default:
    break
  }
}
```

## Event types

- `textDelta` — incremental text tokens.
- `reasoningDelta` — incremental reasoning (if provider supports it).
- `toolCall` / `toolResult` — tool execution lifecycle.
- `messageStart` / `messageEnd` — message boundaries.
- `finish` — final usage and response.
- `error` — fatal errors (stream ends).

All event naming and ordering is aligned with upstream tests.
