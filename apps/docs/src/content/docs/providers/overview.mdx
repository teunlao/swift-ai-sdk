## Usage Pattern

All providers follow a unified API pattern. You can switch providers by changing the import statement and factory function - your application code remains the same:

```swift
import SwiftAISDK
import OpenAIProvider

// Default instance (uses environment variable)
let model = openai("gpt-4")

// Custom configuration
let provider = createOpenAIProvider(
    settings: OpenAIProviderSettings(
        apiKey: "your-api-key",
        organization: "your-org"
    )
)
let model = provider("gpt-4")

// Use with generateText
let result = try await generateText(
    model: model,
    prompt: "Hello, world!"
)
```

Switch to a different provider:

```swift
import SwiftAISDK
import AnthropicProvider

let model = anthropic("claude-3-5-sonnet-20241022")
let result = try await generateText(
    model: model,
    prompt: "Hello, world!"
)
```

<Aside type="tip">
All providers implement the same `LanguageModelV3` protocol, ensuring consistent behavior across different AI services.
</Aside>

## Provider-Specific Features

While the API is unified, some providers offer unique capabilities:

- **xAI Grok** - Reasoning tokens and web search results
- **OpenAI** - Streaming with function calling, audio transcription
- **Anthropic Claude** - Extended context windows, prompt caching
- **Google Gemini** - Multimodal inputs (text, images, video)

See individual provider documentation for specific features and model IDs.
